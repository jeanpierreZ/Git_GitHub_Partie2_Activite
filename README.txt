Exercice de Git_GitHub_Partie2_Activite

Cet exercice a pour objectif de mettre en pratique les notions acquises dans les deux premières parties de ce cours sur Git et Github.

Ne sachant pas coder (snif), je vais vous présenter les 2 autres sens des chats, autres que la vue, l'ouïe, le goût, le toucher, l'odorat.

Pour cela voir les 2 autres fichiers "Organe de Jacobson" et "Organe vestibulaire".
